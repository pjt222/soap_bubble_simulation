## Key Concepts

### Quadratic Differentials
- **Definition**: Mathematical objects on Riemann surfaces of the form $\phi(z)dz^2$
- Encode how surfaces can be glued together
- Natural framework for string field theory vertices
- Determine allowed deformations of string worldsheets

### Closed String Vertices
- **Interaction points** where multiple closed strings meet
- Unlike open strings, closed strings form loops
- Vertices must respect worldsheet modular invariance
- Construction requires careful treatment of string midpoints

### Connection to Minimal Surfaces
Quadratic differentials arise in:
- Teichmüller theory (deformations of Riemann surfaces)
- Minimal surface problems
- Jenkins-Strebel differentials (horizontal trajectories)

## Mathematical Framework

### Quadratic Differential Form
$$\phi = \phi(z) dz^2$$

Under coordinate change $z \to w(z)$:
$$\phi(z) dz^2 = \phi(z) \left(\frac{dz}{dw}\right)^2 dw^2$$

### Horizontal/Vertical Trajectories
- **Horizontal**: curves where $\phi(z)dz^2 > 0$
- **Vertical**: curves where $\phi(z)dz^2 < 0$
- These trajectories define natural coordinates on moduli space

### Jenkins-Strebel Differentials
Special quadratic differentials where:
- Horizontal trajectories form closed curves
- Encode decomposition of surface into cylinders
- Used to construct string vertices
